# 🛠️ Food Shop Admin Panel

This is the admin dashboard for the Food Ordering Web Application.  
It allows administrators to manage food items, users, and orders.

---

## 🚀 Features

- Add new food items
- Delete food items
- View all orders
- change status
- Admin dashboard interface

---

## 🛠️ Tech Stack

- React.js
- JavaScript
- HTML
- CSS

---

## ⚙️ How to Run Locally

### 1️⃣ Clone the repository

git clone https://github.com/Asan0786/food-shop-admin.git

### 2️⃣ Navigate to project

cd food-shop-admin

### 3️⃣ Install dependencies

npm install

### 4️⃣ Start the application

npm start

---

## 🌐 Backend API

Make sure backend is running on:
http://localhost:8080
